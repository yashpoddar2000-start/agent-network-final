### Universal Agents (No Specialization by Content Type)

**We're building UNIVERSAL agents that adapt purely based on data** - no specialized "gap analysis" or "paradox hunter" agents, just universal research functions that let the DATA decide where to pivot (like RecurSearch's pattern).

**These universal agents use only two tools (exaAnswerTool + exaDeepResearchTool) with data-driven decision making** - if KFC data reveals boring insights, agents automatically pivot to investigate same-parent dynamics with Taco Bell, or abandon KFC entirely for Wingstop.

### The Beautiful Irony: Universal Function + Expert Identity

**While functionally universal, each agent has deep QSR industry expertise personas** (Wharton MBA analysts, PE advisors, McKinsey consultants) - so they're universal research agents specialized for restaurant industry intelligence.

This creates the perfect balance:
- **HIGH-LEVEL ARCHITECTURE:** Universal (can research ANY QSR company, ANY insight type)  
- **INDIVIDUAL AGENT IDENTITY:** QSR Expert Specialists with institutional credibility

### Quality Through Iron Sharpening Iron + Pre-Filtering

**Quality ensured through dual-layer approach:**
1. **Topic Pre-Filtering:** QSR Topic Picker Agent ensures only high-viral-potential research targets reach core network
2. **Institutional Evaluation:** QSR experts create insights for brutal evaluation by "Senior Restaurant Analyst" - iron sharpening iron until only McKinsey-grade insights pass through

**Result: Efficient resource allocation + institutional quality standards = systematic viral insight generation**
